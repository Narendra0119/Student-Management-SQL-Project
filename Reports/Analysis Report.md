1️⃣ Show All Student Details

Purpose:
Displays complete student data for inspection and validation.

Insight:
Helps analysts understand the dataset structure and values.

2️⃣ Average Score in Each Subject

Purpose:
Calculates overall class performance in Math, Science, and English using AVG().

Insight:
Identifies strongest and weakest subjects across all students.

3️⃣ Top Performer

Purpose:
Finds the student with the highest combined score.

Logic:
Math + Science + English → TotalScore
Sorted in descending order, top record selected.

Insight:
Useful for merit analysis and ranking.

4️⃣ Count of Students per Grade

Purpose:
Uses GROUP BY to count students in each grade.

Insight:
Shows grade distribution and academic performance spread.

5️⃣ Average Score by Gender

Purpose:
Analyzes subject-wise performance differences between genders.

Insight:
Helps in demographic-based academic analysis.

6️⃣ Students with Math Score > 80

Purpose:
Filters high-performing students using WHERE.

Insight:
Useful for identifying subject toppers or eligibility checks.

Insight:
